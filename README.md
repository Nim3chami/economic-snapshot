# Economic Dashboard Web Application

## Overview
This is a **web-based interactive dashboard** that visualizes real-time and historical economic data of countries. The system integrates public APIs to fetch country metadata and economic indicators, providing users with a clear, visually appealing, and responsive interface to analyze economic trends.

The dashboard displays country-specific data such as GDP, inflation, unemployment, interest rates, government debt, exports, imports, government revenue, and health expenditure. Historical trends are rendered in **interactive charts** for comparison and analysis.

---

## Features
- Search for any country and view its latest economic indicators.
- Display historical trends of major economic indicators using interactive charts.
- View country metadata including population, capital, currency, and flag.
- Fully **client-side implementation**, fetching data from public APIs (REST Countries API and World Bank API).
- Responsive design, accessible on desktops, tablets, and mobile devices.
- Error handling for invalid or missing data.

## Technology Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6)  
- **Charting Library:** Chart.js for interactive visualizations  
- **APIs Used:**  
  - [REST Countries API](https://restcountries.com/) – for country metadata  
  - [World Bank API](https://datahelpdesk.worldbank.org/) – for economic indicators
